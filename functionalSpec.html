Music Paradise

Purpose

The main purpose of this website is to educate people about different types of musical instruments. People should be able to view different muscial instruments based on the category selected. For example, selecting a 'string instrument' category should show different string instruments like violin, guitar, ukulele and more.

Objectives

  • Information: To provide knowledge about different types of musical instruments
  • Price: Price range of different musical instruments
  • UI design: Attractive and easy to use
  • Sign up form: User will be able to sign up and receive notifications
  • Customer survey: Must have a 70% satisfaction rating
  • Discussion forum: User will be able to ask questions and comment on other's question

Primary Audience

The website's main focus will be on any individual who plays a musical instrument, or are learning any musical instrument. The sign up form will collect information from user that will let us know if the user plays any musical instrument.

Secondary Audience

This site will also target any children of age ranging from 10 to 18 who are interested in learning a musical instrument. Additionally it will also target on any individual with previous music experience.
